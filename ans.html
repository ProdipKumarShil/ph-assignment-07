Que:1 Difference between Map Filter forEach and Find

  • Map Map is a callback function. It takes 3 parameters Element index and array. It execute every elements of array. and return an array.
  • Filter: Filter returns an array. It find specific String, Number.
  • forEach: It is a function. forEach apply only in an Array. It returns single elements of an array.
  • Find:Find function search single element of an array. Find search by its condition.

Que:2 Difference between Local storage Session storage and Cookies

  • Local Storage: It is a property. it allow you to store data on storage. If you close tab, browser or shutdown your computer this data doesn't delete.
  • Session Storage: This is like local storage. If you close tab. It remove all data form storage.
  • Cookies: It is a amount of info that persist client side and server side.
